# React-Native-Wechat

[React Native] bridging library that integrates WeChat SDKs:

- [x] iOS SDK 1.8.6 包含支付功能版本，支持跳转微信小程序
- [x] Android SDK 221

## Installation

```sh
$ npm install txh-react-native-wechat --save
```
